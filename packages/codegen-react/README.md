# @zaikit/codegen-react

Code generation tool that scans your agent's tool definitions and generates React tool renderer boilerplate.

## Install

```bash
pnpm add -D @zaikit/codegen-react
```

## Usage

```bash
npx zaikit-codegen-react --agent ./src/agent.ts --output ./src/tools
```
