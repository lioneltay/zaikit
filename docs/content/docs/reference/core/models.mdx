`@zaikit/core` provides a few pre-configured model instances and re-exports commonly used utilities from the AI SDK and internal modules.

```ts
import { model, fastModel, proModel } from "@zaikit/core";
```

## Pre-configured Models

These are convenience exports that use Google Vertex AI. They are optional -- you can use any AI SDK `LanguageModel`.

```ts
import { createVertex } from "@ai-sdk/google-vertex";

const vertex = createVertex({
  project: process.env.GOOGLE_CLOUD_PROJECT,
  location: process.env.GOOGLE_CLOUD_LOCATION,
});

export const model: LanguageModel = vertex("gemini-3-flash-preview");
export const fastModel: LanguageModel = vertex("gemini-3-flash-preview");
export const proModel: LanguageModel = vertex("gemini-3.1-pro-preview");
```

| Export | Model | Use case |
| --- | --- | --- |
| `model` | `gemini-3-flash-preview` | Default model. Good balance of speed and quality. |
| `fastModel` | `gemini-3-flash-preview` | Alias for the fast model. |
| `proModel` | `gemini-3.1-pro-preview` | Higher quality model for complex tasks. |

## Using Other Providers

`createAgent` accepts any AI SDK `LanguageModel`. You are not limited to the pre-configured exports:

```ts
import { createAgent } from "@zaikit/core";
import { openai } from "@ai-sdk/openai";

const agent = createAgent({
  model: openai("gpt-4o"),
  tools: { /* ... */ },
});
```

```ts
import { anthropic } from "@ai-sdk/anthropic";

const agent = createAgent({
  model: anthropic("claude-sonnet-4-20250514"),
  tools: { /* ... */ },
});
```

```ts
import { createVertex } from "@ai-sdk/google-vertex";

const vertex = createVertex({ project: "my-project", location: "us-central1" });

const agent = createAgent({
  model: vertex("gemini-2.5-pro"),
  tools: { /* ... */ },
});
```

## Re-exports from AI SDK

These are re-exported from the `ai` package for convenience:

| Export | Source | Description |
| --- | --- | --- |
| `streamText` | `ai` | Stream text from a language model. Used internally by `createAgent`. |
| `convertToModelMessages` | `ai` | Convert `UIMessage[]` to the format expected by model providers. |

## Re-exports from Internal Modules

| Export | Type | Description |
| --- | --- | --- |
| `hasSuspendedTool` | `StopCondition<any>` | Stop condition that halts the LLM loop when any tool returned a `SuspendResult`. Always composed into `createAgent`'s `stopWhen`. |
| `isSuspendResult` | `(value: unknown) => value is SuspendResult<unknown>` | Type guard that checks if a value is a `SuspendResult`. |
| `SuspendResult` | Type | The type returned by the `suspend()` function inside suspendable tools. |
| `runWithSuspendContext` | `<T>(context: { resumeData?: unknown }, fn: () => T) => T` | Run a function with resume data available via `AsyncLocalStorage`. Used internally by the agent during resume. |
| `getResumeData` | `<T>() => T \| undefined` | Read the current resume data from `AsyncLocalStorage`. Used internally by `createTool`. |

```ts

// Agent

// Models
export { fastModel, model, proModel } from "./model";

// Suspend
export { hasSuspendedTool } from "./stop-conditions";

export { isSuspendResult } from "./suspend";

```
